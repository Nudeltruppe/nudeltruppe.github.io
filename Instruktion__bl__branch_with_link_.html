Instruktion: bl (branch with link)

Die instruktion bl wird benutzt um von einem codeblock zu einem unterprogramm zu springen. Bei der ausfürung wird das link register(lr) gesetzt was heist, dass return mit bx lr möglich ist.

subroutine: <-----|
  bx lr---------------------|
                  |         |
main:             |         |
  bl subroutine --|         |
  // andere instruktionen <-|
